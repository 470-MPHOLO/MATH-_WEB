Arithmetic Operations

Addition
Need help?

To add numbers:

  1. Separate numbers with commas
  2. The calculator will sum all numbers together
  3. Example: 5, 10, 15 = 5 + 10 + 15 = 30
Subtraction
Need help?

To subtract numbers:

  1. The first number is the starting value
  2. All subsequent numbers are subtracted from it
  3. Example: 20, 5, 3 = 20 - 5 - 3 = 12
Multiplication
Need help?

To multiply numbers:

  1. All numbers are multiplied together
  2. Example: 2, 3, 4 = 2 × 3 × 4 = 24
Division
Need help?

To divide numbers:

  1. The first number is the dividend
  2. All subsequent numbers are divisors
  3. Example: 100, 5, 2 = 100 ÷ 5 ÷ 2 = 10
  4. Division by zero is not allowed

Algebra Calculator

Linear Equation Solver
Need help?

To solve a linear equation ax + b = c:

  1. Subtract b from both sides: ax = c - b
  2. Divide both sides by a: x = (c - b) / a
Quadratic Equation Solver
Need help?

To solve a quadratic equation ax² + bx + c = 0:

  1. Calculate the discriminant: D = b² - 4ac
  2. If D > 0: Two real solutions: x = (-b ± √D) / (2a)
  3. If D = 0: One real solution: x = -b / (2a)
  4. If D < 0: Two complex solutions
System of Equations
Need help?

To solve a system of equations:

  1. Calculate the determinant: D = a₁b₂ - a₂b₁
  2. If D ≠ 0, unique solution exists
  3. x = (c₁b₂ - c₂b₁) / D
  4. y = (a₁c₂ - a₂c₁) / D

Matrix Calculator

Matrix Addition
Need help?

To add matrices:

  1. Add corresponding elements
  2. A + B = [a₁₁+b₁₁, a₁₂+b₁₂; a₂₁+b₂₁, a₂₂+b₂₂]
Matrix Multiplication
Need help?

To multiply matrices:

  1. Multiply rows of A by columns of B
  2. Element (i,j) = sum of (row i of A) × (column j of B)
Matrix Determinant
Need help?

To calculate determinant of 2x2 matrix:

  1. det(A) = a₁₁ × a₂₂ - a₁₂ × a₂₁
  2. Example: det([[a,b],[c,d]]) = ad - bc
